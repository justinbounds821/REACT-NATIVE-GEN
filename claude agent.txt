 Cum funcționează Claude ca agent de codare autonom 🧠
Claude Opus/Sonnet sunt modele LLM care pot înțelege proiecte complexe, naviga prin fișiere și genera cod structurat pentru ore întregi (ex: 7h cu Rocketin). Pot gestiona „to-do lists” și își auto-întrețin progresul pas cu pas.

Elementele cheie:

Files API – acces la fișierele proiectului, citire/scriere de cod și structuri de directoare.

Code Execution Tool – posibilitatea de a rula scripturi, testa și corecta cod în ciclu automat.

Web Search / Citations – pentru a încorpora informații actualizate.

MCP (Model Context Protocol) – conectează modelul la sisteme business via API contextuale (ex: Thomson Reuters).

2. Cum construim un agent de codare Claude + n8n
🔧 Setup tehnic
Claude API – obții cheie API de la Anthropic.

n8n – platformă de workflow no-code/low-code care rulează pe Docker/VPS. Permite crearea de fluxuri automate.

Mediu unde rulăm și testăm cod – cont Docker sau VM; CLI (Node.js, Python etc.).

📌 Flux de lucru în n8n
Trigger – inițiat manual, periodic (cron) sau prin webhook (ex: GitHub event).

HTTP Request către Claude:

Include fișierele relevante (ex. cod curent, package.json, README).

Prompt extins, structurat: „Analizează repo, planifică-o, scrie funcționalitatea X, generează cod și teste.”

Claude returnează:

Lista de taskuri („to-dos”),

Cod sursă,

Comenzi de testare/compilare.

Write Files – scriem fișierele generate direct în repo local.

Execute Code Tool – comandă shell prin nod n8n: npm install, npm test, pytest etc.

Log + Feedback – colectăm output de execuție și îl trimitem înapoi la Claude pentru ajustări, în ciclu.

🧩 Model de prompt:
text
Copy code
1. Citește cele 3 fișiere atașate.
2. Analizează și planifică task-uri: create user endpoint, adaugă test etc.
3. Trimite code: fișiere noi modificările plus comenzi de test și compilare.
3. Cum rulăm totul autonom
n8n rulează fluxul periodic sau la trigger extern (ex: issue pe GitHub).

Claude primește contextul actualizat și generează cod nou.

Codul este scris, compilat/testat automat.

Rezultatele sunt logate și, dacă e nevoie, Claude primește feedback și rafinează.
